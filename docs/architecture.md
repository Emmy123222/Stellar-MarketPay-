## Security Model

| Concern | Mitigation |
|---------|-----------|
| Payment disputes | Soroban contract enforces rules вҖ” no human intermediary |
| Key exposure | Freighter signs locally вҖ” private key never leaves browser |
| Fake job postings | Wallet signature required to post (v1.1) |
| Double spending | Stellar sequence numbers prevent replay |
| Sybil freelancers | Reputation system planned (v1.4) |
| Backend trust | Backend is stateless helper вҖ” all payments are on-chain |
